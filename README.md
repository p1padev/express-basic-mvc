# 📝 Message Board

A simple message board application built with Express and TypeScript.

## 🚀 How to Run Locally

1. Clone the repository
2. Install dependencies:
   ```
   pnpm install
   ```
3. Run in development mode:
   ```
   pnpm dev
   ```
4. Build for production:
   ```
   pnpm build
   ```
5. Start production server:
   ```
   pnpm start
   ```

## 🛠️ Technologies Used

- Express.js
- TypeScript
- EJS (Embedded JavaScript templating)
- Pico CSS

## 📦 Package Information

For more details on dependencies and scripts, see `package.json` in the project root.

## 🔧 TypeScript Configuration

TypeScript is configured for a Node.js environment. For full configuration details, check the `tsconfig.json` file in the project root.
